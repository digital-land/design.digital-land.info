How to publish your data

The DLUHC service, RIPA, needs machine readable data about planning and housing to help their users submit valid planning applications.

There is an opportunity for Digital Land to provide RIPA with this data whilst learning about the barriers local authorities face in publishing data and what Digital Land needs to do to reliably collect data and provide it to RIPA (and other services in the future).

Version 1 - basic page

  • A plain content version of the guidance

Version 1B - Enhanced fields display

  • Uses the GOVUK Design System 'Summary List' component to display the data specifications
  • Adds a 'contents' list at the top
  • Adds a 'back to top' link at the bottom

Version 2 - Tabular field reference matrix

  • Displays the dataset specifications in a table with reference of field definitions

Version 3 - Visual tabular fields reference matrix

  • Alternate version of the table matrix using iconography

Version 4 - Summary list with jump links

  • Revises content from the user perpective
  • Adds back to top links after each dataset
  • Adds support information to aside column
