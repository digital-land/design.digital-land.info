Prototypes

The following links are to individual prototypes and page designs that were assembled and researched by the Planning Data team of The Department for Levelling Up, Housing and Communities.

Website

Prototypes for the website which will eventually become planning.data.gov.uk

  • 'Entity' page
    Prototype designs of webpages intended to show and help users to understand individual entities.
  • How to publish your data
    Prototype designs of guidance for Local Planning Authorities intended to help them publish data required by planning.data.gov.uk.

Feedback

Prototypes for the application developed to provide feedback to data providers such as local authorities

  • Feedback Application
    Page designs for feedback.digital-land.info

  • LPA Feedback Prototype (opens in a new window)
    Figma prototype of early sketches to show feedback to Local Planning Authorities about the data DLUHC has collected from them.

Performance

Prototypes for the application developed to help track and manage the quality of planning data

  • Performance grid
    Design experiment for showing health of datasets for RIPA.

Other prototypes

  • Housing numbers (opens in a new window)
    Experimental work on collection of housing numbers from local plans.
  • Performance dashboard (opens in a new window)
    Prototype dashboard to monitor the performance of the planning data platform.
  • Brownfield sites validator (opens in a new window)
    Prototype data validation for Brownfield sites.
  • The code for these prototypes on Github
